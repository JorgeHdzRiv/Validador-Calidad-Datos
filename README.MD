# Validador Automático de Calidad de Datos

Aplicación web diseñada para automatizar la auditoría técnica de datasets (CSV/Excel). Desarrollada con Python, Pandas y Streamlit.

## 🚀 Características
- **Detección de Anomalías:** Identificación de nulos, duplicados y tipos de datos inconsistentes.
- **Reporte Visual:** Dashboard interactivo para análisis exploratorio rápido.
- **Exportación:** Generación de reportes de calidad descargables.
- **Modular:** Código estructurado para fácil escalabilidad.

## 🛠 Stack Tecnológico
- **Lenguaje:** Python 3.x
- **Framework:** Streamlit
- **Análisis:** Pandas
- **UI:** Integración nativa (Theme-aware)

## 📦 Instalación
1. Clona el repo: `git clone [url-de-tu-repo]`
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta la app: `streamlit run app.py`
